# 📱 React Native Store App

A simple React Native e-commerce mobile app built with Expo.\
It includes product listing, product details, and basic navigation.

------------------------------------------------------------------------

# ⚙️ Features

-   Product listing screen
-   Product detail screen
-   Local image support
-   React Navigation setup
-   Clean UI design

------------------------------------------------------------------------

# 🧰 Prerequisites

Before running this project, install the following:

## 1. Install Node.js (LTS)

https://nodejs.org

## 2. Install Expo Go (Mobile App)

Available on Play Store / App Store

------------------------------------------------------------------------

# 🔍 Check Installation

``` bash
node -v
npm -v
```

------------------------------------------------------------------------

# 📥 Clone Repository

``` bash
git clone https://github.com/bilawal-mahmood/react-native-store-app.git
cd react-native-store-app
```

OR download ZIP and extract it.

------------------------------------------------------------------------

# 📦 Install Dependencies

``` bash
npm install
```

------------------------------------------------------------------------

# 🚀 Run Project

``` bash
npx expo start
```

------------------------------------------------------------------------

# 📱 Run on Mobile

1.  Open Expo Go app\
2.  Scan QR code\
3.  App runs instantly

------------------------------------------------------------------------

# ❗ Important Notes

-   node_modules is NOT uploaded to GitHub
-   Always run npm install after cloning
-   If issue occurs:

``` bash
rm -rf node_modules
npm install
```

------------------------------------------------------------------------

# 🧠 Clear Cache

``` bash
npx expo start -c
```
